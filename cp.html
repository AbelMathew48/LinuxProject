cp

The cp command is used to copy files and directories in Linux. It allows you to duplicate files or entire directories from one location to another. By default, cp does not copy directories unless the -r (recursive) option is used.

Copy a file:
cp source.txt destination.txt
Copy a file to a directory:
cp file.txt /path/to/directory/
Copy a directory and its contents:
cp -r folder_name /path/to/destination/
Terminal Representation
student@mca-System-Product-Name:~/Desktop$ cp source.txt destination.txt
student@mca-System-Product-Name:~/Desktop$ cp file.txt /path/to/directory/
student@mca-System-Product-Name:~/Desktop$ cp -r folder_name /path/to/destination/
