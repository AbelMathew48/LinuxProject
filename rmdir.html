rmdir/rm

The rmdir command is used to remove empty directories in Linux. If the directory contains files or subdirectories, it cannot be removed using rmdir. Instead, you can use the rm command to remove files and directories, including non-empty ones. Use the -r option with rm to remove directories and their contents recursively.

Remove an empty directory:
rmdir empty_folder
Remove a file:
rm file.txt
Remove a directory and its contents recursively:
rm -r folder_name
Terminal Representation
student@mca-System-Product-Name:~/Desktop$ rmdir empty_folder
student@mca-System-Product-Name:~/Desktop$ rm file.txt
student@mca-System-Product-Name:~/Desktop$ rm -r folder_name
