mkdir

The mkdir command is used to create new directories in Linux. You can specify the name of the directory to be created, and it will be added to the current directory. Optionally, you can specify a complete path to create a directory at a different location.

student@mca-System-Product-Name
mkdir new_folder
Action: Creates a directory named new_folder in the current working directory.
Terminal Representation
student@mca-System-Product-Name:~/Desktop$ mkdir new_folder
student@mca-System-Product-Name:~/Desktop$ ls
existing_file.txt new_folder
