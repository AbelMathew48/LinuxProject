grep

The grep command is used to search for specific patterns within files in Linux. It supports regular expressions and provides options for case sensitivity, recursive search, and more.

Search for a word in a file:
grep "pattern" filename
Search recursively in directories:
grep -r "pattern" directory/
Ignore case sensitivity:
grep -i "pattern" filename
Terminal Representation
student@mca-System-Product-Name:~/Desktop$ grep "hello" file.txt
Hello, World!
This is a hello example.
student@mca-System-Product-Name:~/Desktop$ grep -i "example" file.txt
