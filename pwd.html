pwd

The pwd (print working directory) command is used to display the absolute path of the current working directory. It shows the full directory path from the root directory to the current directory where the user is located. This command is helpful to determine your current location in the file system.

student@mca-System-Product-Name
pwd
Output: /home/student/Desktop
Terminal Representation
student@mca-System-Product-Name:~/Desktop$ pwd
/home/student/Desktop
student@mca-System-Product-Name:~/Desktop$
